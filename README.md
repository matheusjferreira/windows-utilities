> # Windows utilities

> :warning: For Windows 10 and Windows 11.

> ## Tre Project:

Simple utilities to prepare the windows environment for use, focusing on ease and performance.

> ## What will be changed?

* Context menu will be the same as in Windows 10 (only for Windows 11);
* Background apps will be disabled;
* Quick access will be removed;
* Maximum performance power option will be released;
